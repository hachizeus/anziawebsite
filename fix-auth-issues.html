Quick Fixes

Manual Steps

1. Clear Browser Data

If the automatic fixes don't work, manually clear your browser data:

  • Press F12 to open Developer Tools
  • Go to Application/Storage tab
  • Clear Local Storage and Session Storage
  • Refresh the page

2. Console Commands

You can also run these commands in the browser console (F12 ‚Üí Console):

// Clear all authentication data localStorage.clear(); sessionStorage.clear(); location.reload();

3. If Still Having Issues

  • Try using an incognito/private browser window
  • Clear your browser cache completely
  • Try a different browser
  • Wait 10 minutes and try again (rate limits will reset)
